Modern User Interface

by PeterBitt
--------------------------




This mod overhaules the user interface in Morrowind by trying to give it a more modern touch, while keeping the characteristic parts vanilla.


Features:

- flat, simple, minimalistic design
- all borders and other lines are now dark-grey
- all dark-grey parts are slightly transparent, this effect scales with ingame UI opacity slider
- interactive elements and the headers keep their vanilla look
- the compass arrow in map screen is smaller
- detect spell map marker are filled circles instead of only outlines
- door marker are filled boxes instead of only outlines
- health, magica, etc bars are flat
- vanilla enchantment icon is more transparent




Installation:
-------------------------

1. copy the "Textures" folder in your "Morrowind/Data Files" folder
3. play the game
