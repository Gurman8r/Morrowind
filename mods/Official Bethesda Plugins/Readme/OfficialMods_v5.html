INSTALLATION:
If (for example) you have Morrowind installed on C:\Program Files\Bethesda Softworks\Morrowind then all you have to do is place a check next to "Use Folder Names" and then extract the contents of this zip to C:\Program Files\ - everything should go into the proper directories from there. Remember to substitute C:\Program Files\ for your actual install directory!
