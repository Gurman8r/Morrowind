**README**

- STEP 1 - If you are NOT using True Lights and Darkness, skip to STEP 4 - 

- If you ARE using True Lights and Darkness, the STEP 2 settings are required but STEP 1 
is more of a recommendation -

In your settings.cfg file, find or create a line at the top called [Shaders]
underneath it, make sure these 3 lines are included:

force shaders = true
force per pixel lighting = true
clamp lighting = false

If you want to revert back to default behavior, these lines should be like this:

force shaders = false
force per pixel lighting = false
clamp lighting = true

You can now save this file and close it.

- STEP 2 - 

Open your openmw.cfg file and find the "LightAttenuation" lines, at the top.

These are the DEFAULT settings:

fallback=LightAttenuation_UseConstant,0
fallback=LightAttenuation_ConstantValue,0.0
fallback=LightAttenuation_UseLinear,1
fallback=LightAttenuation_LinearMethod,1
fallback=LightAttenuation_LinearValue,3.0
fallback=LightAttenuation_LinearRadiusMult,1.0
fallback=LightAttenuation_UseQuadratic,0
fallback=LightAttenuation_QuadraticMethod,2
fallback=LightAttenuation_QuadraticValue,16.0
fallback=LightAttenuation_QuadraticRadiusMult,1.0
fallback=LightAttenuation_OutQuadInLin,0

REPLACE them with these:

fallback=LightAttenuation_UseConstant,0
fallback=LightAttenuation_ConstantValue,0.0
fallback=LightAttenuation_UseLinear,0
fallback=LightAttenuation_LinearMethod,1
fallback=LightAttenuation_LinearValue,3.0
fallback=LightAttenuation_LinearRadiusMult,1.0
fallback=LightAttenuation_UseQuadratic,1
fallback=LightAttenuation_QuadraticMethod,2
fallback=LightAttenuation_QuadraticValue,16.0
fallback=LightAttenuation_QuadraticRadiusMult,1.6
fallback=LightAttenuation_OutQuadInLin,0

- STEP 3 - 

When using Per Pixel Lighting, you will need a plugin that removes Morrowind's negative lights

Download and install PeterBitt's Negative Lights Remover here:

https://www.nexusmods.com/morrowind/mods/42306/?tab=files&navtag=http%3A%2F%2Fwww.nexusmods.com%2Fmorrowind%2Fajax%2Fmodfiles%2F%3Fid%3D42306&pUp=1

- STEP 4 -

Paste the weather settings below BETWEEN these lines:

fallback=Weather_Clear_Transition_Delta,.015

and

fallback=Moons_Secunda_Size,40

overwriting what was there.
