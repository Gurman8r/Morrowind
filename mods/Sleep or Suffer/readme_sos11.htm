Description
Sleep is now a necessity. If you go too long without sleep, your character is definately going to feel the effects of sleep deprivation. Try and see how long you can stay awake and survive. Stay awake for 132 hours or more and you enter the Realm of Chaos!
Installation and use
  1. Unzip the archive into Morrowind's data files folder.
  2. From the Morrowind Launcher, select the appropriate plugin to load: Either SleepOrSuffer.esp or SleepOrSuffer-Tribunal.esp.
To see how long you've been awake and how long you must sleep to get rid of all deprivation effects, equip the Ring of the Sandman. This ring is added to your inventory upon loading this plugin.

If you're going to use the non-Tribunal version of this plugin with a savegame already in progress, travel to Seyda Neen to have the plugin activated.

Important note:
It's possible that the effects from this plugin will lower your agility, endurance, strength, and willpower to less than zero (your character sheet will show zero though). These attributes determine your maximum fatigue, which will be zero if the above attributes are all zero. The most likely time this will happen is if you reach the Realm of Chaos deprivation level. It's possible for this to happen at other deprivation levels, but not nearly as likely.
If you're using my other plugin, Falling Down, this will make you fall down and unable to get up. You will still be able to look up and down and access your inventory, so there is a way out...
If you're not using Falling Down, you won't fall down and you will still be able to move around provided you drop enough items from your inventory so that your current encumbrance is zero. Encumbrance is five times your strength.
My suggestion is to be certain you have some way to counter the effects this plugin induces...

Plugin history and sleep deprivation effects
Version 1.0
    • Added one global script, _sos_main.
    • Added two global variables, _sos_gh_total and _sos_gh_reftime.
    • Added _sos_main to the start script list for the Tribunal version.
    • Added one activator, _sos_start_activator, to cell (-2,-9), Seyda Neen for the non-Tribunal version.
    • Added script, _sos_startscript, attached to _sos_start_activator in order to start _sos_main for the non-Tribunal version.
    • Added 10 spell ability effects for the 10 levels of sleep deprivation. See the table below.
    • Added Ring of the Sandman, _sos_totalrefreshment and attached script _sos_trinfo, to the player's inventory upon loading this plugin. It will tell the player how long he/she has been awake and also the required amount of sleep for total refreshment.
Version 1.1
    • Added one more sleep deprivation level, severe.
    • Renamed Virge of Death to Insane.
    • Changed each deprivation level length from 8 gamehours to 10 gamehours.
    • Replaced death with another deprivation level, Realm of Chaos, and it occurs at 132 gamehours.
    • Changed the effects for each deprivation level to account for the above changes.
    • If you're a vampire, you won't be affected by sleep deprivation.

Compatibility
This plugin is compatible with any other plugin available. However, if there are other plugins similar to this one (i.e. causing the player negative effects from lack of sleep) and they are used at the same time, too many negative effects and/or strange occurrences may result.
