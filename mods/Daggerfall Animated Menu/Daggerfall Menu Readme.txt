The Elder Scrolls III MORROWIND:

==================
Daggerfall Menu
==================

by vtastek
28/04/2012

1. DESCRIPTION

Changes Morrowind's background menu with a video that mimics Daggerfall's main menu background.

2. Requirements

MGE XE

3. INSTALLING THE PLUGIN

Just install like any mod. The file will go into data/videos folder.



You're free to do anything with this mod. Enjoy!
